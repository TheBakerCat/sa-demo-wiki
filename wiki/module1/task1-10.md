# 10. Настройте инфраструктуру разрешения доменных имён для офисов HQ и BR:

• Основной DNS-сервер реализован на HQ-SRV

• Сервер должен обеспечивать разрешение имён в сетевые адреса
устройств и обратно в соответствии с таблицей 3

• В качестве DNS сервера пересылки используйте любой общедоступный
DNS сервер(77.88.8.7, 77.88.8.3 или другие)

Устонавливаем bind
```
apt-get install bind bind-utils
```
Настраиваем /var/lib/bind/etc/bind/options.conf 
(Так же находиться по пути /etc/bind/options.conf)

Потребуеться дописать или раскоментить следующее:
```
listen-on { 127.0.0.1; 192.168."номер vlan".0/"маска от HQ-CLI" (если не работает - писать "ip адрес HQ-CLI" без маски) };

forwarders { 8.8.8.8; };

recursion yes;

allow-query { 192.168."номер vlan".0/"маска от HQ-CLI"; 127.0.0.1; };

dnssec-validation no;
```
