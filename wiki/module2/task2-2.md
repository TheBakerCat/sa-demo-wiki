## 2. Сконфигурируйте файловое хранилище на сервере HQ-SRV:

::: details Задания

• При помощи двух подключенных к серверу дополнительных дисков
размером 1 Гб сконфигурируйте дисковый массив уровня 0

• Имя устройства – md0, при необходимости конфигурация массива
размещается в файле /etc/mdadm.conf

• Создайте раздел, отформатируйте раздел, в качестве файловой системы
используйте ext4

• Обеспечьте автоматическое монтирование в папку /raid

:::

### 0. Немного советов

lsblk(расшифровывается как «list block devices» — «список блочных устройств») -
выводит диски в системе и их точки монтирования

Прописав команду mount -a, можно проверить правильно ли написан fstab 

### 1. Конфиигурация файловое хранилище

Устанавливаем mdadm (от англ. multiple devices driver)

```
apt-get install –y mdadm
```
Создаём массив

```
mdadm --create /dev/md0 –l 0 –n 2 /dev/sdb /dev/sdc
```

Где:

/dev/md0 - название для логического девайса рейда

-l 0 – уровень RAID;

-n 2 – количество дисков, из которых собирается массив;

/dev/sdb /dev/sdc - диски для рейда


Форматируем раздел

```
mkfs.ext4 /dev/md0
```

Создаём директорию

```
mkdir /raid
```

Монтируем рейд в директорию /raid 

```
mount /dev/md0  /raid
```


Редакируем файл /etc/fstab, добовляя следуйщие строчки:

```
/dev/md0    /raid    ext4    defaults    0    0
```
